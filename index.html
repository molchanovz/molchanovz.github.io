Об общежитии

Студенческое общежитие РГГУ расположено в Южном административном округе г. Москвы по адресу ул. Кировоградская д. 25, корп. 1 (на расстоянии 600 метров от станции метро «Улица Академика Янгеля»). Общежитие блочного типа. Каждый блок состоит из двухместной и трехместной комнаты. В блоке есть раздельный санузел, а также бесплатная wi-fi сеть для доступа в Интернет. На каждом этаже оборудована общая кухня. Сведения об общежитии в соответствии с требованиями, предусмотренными Приказом Рособрнадзора от 14 августа 2020 года №831 "Об утверждении Требований к структуре официального сайта образовательной организации в информационно-телекоммуникационной сети "Интернет" и формату представления информации".

Текст по центру

Для оплаты проживания в общежитии необходимо: 1. Обратиться в Управление по работе со студентами для расчета стоимости проживания (Миусская пл., д.6, корп.7, каб.226). 2. Способы оплаты: - через приложение Сбербанк Онлайн (Инструкция по оплате через Сбербанк Онлайн 2018) - комиссия 1%; - на сайте РГГУ: раздел Оплатить проживание - оплата без комиссии; - через терминал «Уралсиб» (Инструкция по оплате через терминал «Уралсиб» 2017), ближайший терминал установлен по адресу: Миусская пл., д.6, корп.6 в холле первого этажа - комиссия 0,1%. 3. Отправить чек для подтверждения оплаты на почту hostel@rggu.ru. Приказы о размере платы за проживания можно опубликованы в разделе "Нормативные документы".
